iterative-avgstd

Iterative calculation of mean and standard deviation

I have developed this tiny library to calculate simple statistics on analog samplings for my Arduino project without the need to store the whole amount of data. Details on the algorithm in the docs folder.

Here is how it started...

Some time ago I was sick with some fever. The only thing I could do was to lay in bed. I took an electronic thermometer and measured my temperature, say 37.8C. It took so little time that I tried again, 37.4C. I tried a third time and the temperature kept changing. I thought to measure a few times and then to calculate the average, to see a more stable result.

After a few sampling, doing it by heart without a piece of paper starts to be a bit complex. I started thinking that there must be a way to use the latest measured average and update it with the next sample.

I worked out a simple formula:

N = 1/N * sum_i^N T_i (N+1) = 1/(N+1) * sum_i^(N+1) T_i = 1/(N+1) * ( sum_i^N T_i + T_(N+1))

(N+1) = N/(N+1) * _N + T(N+1)/(N+1)

So the next average can be calculated by keeping in mind only 3 numbers: the average, the number of samplings and the last sample. That's nice! I worked out also the rule for the standard deviation, or better for the variance.

var_(N+1) = var_N * ((N-1)/N) + (T_(N+1) - _N)^2

Here, you need to remember only 4 numbers, the average, the variance, the sample and the number of samplings.
